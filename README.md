# openDAQ-CI-sandbox
Sandbox for developing reusable CI/CD workflows for [openDAQ](https://github.com/openDAQ/openDAQ)

## Project Purpose

This repository serves as a sandbox for:
- **Developing** modular reusable workflows for openDAQ
- **Testing** CI/CD changes in an isolated environment
- **Refactoring** existing CI infrastructure
- **Prototyping** new automations and integrations
